# ISTQB CTFL 모바일 모의고사

ISTQB CTFL v4.0.1 학습을 위한 비공식 개인용 모의고사입니다.

- 25회차 선택 화면(실전 검수형 2회 + 공식 한국어 Sample A~D 실전 + 변형 실전)
- 공식 배분과 같은 40문제 / 60분 / K1 8·K2 24·K3 8
- 장별 선택, 10·20·40문제 랜덤 연습, 저장된 오답 재시험
- 단일·복수 정답 문항 지원
- 모바일 답안 자동 저장 및 이어 풀기
- 즉시 채점, 영역별 점수, 오답 해설
- 결과 일괄 공유 및 Markdown 저장

자체 제작 회차는 CTFL v4.0.1 실러버스 학습 목표를 바탕으로 작성했습니다. 3·9·16·23회차는 KSTQB가 공개한 공식 한국어 Sample A·B·C·D 문제와 해설을 출처로 사용하며 저작권은 KSTQB와 ISTQB에 있습니다.
